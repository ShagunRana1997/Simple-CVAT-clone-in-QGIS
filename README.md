# Simple-CVAT-clone-in-QGIS

A Simple CVAT style Annotation plugin to annotate images in similar style like in CVAT.
**Features:**
1. Load multiple images and scroll between them left to right just like in CVAT.
2. Create your own Annotation labels and attributes or just use label format from your CVAT project.
3. Save your annotations in YoloV8 format or in CVAT XML format(Bonus Point: Also can save .shp of the annotations).
4. CVAT but better with all the plus points of QGIS. Ability to annotate directly on 16 Bit Tiff. Best of you are training model on Geospatial Data.

**How to Install the Plugin**
1. Download whole repo as a zip file.
2. Open Plugin Manager: In QGIS, navigate to the Plugins menu and select Manage and Install Plugins.
3. In the window that opens, click the Install from ZIP tab on the left sidebar.
4. Click the three dots (...) button to browse your computer and select your .zip file.
5. Click Install Plugin. You may see a security warning if the plugin is from an unofficial source; click "Yes" to proceed.
